# 可燃冰机器人科学平台

## 概述
可燃冰机器人科学平台是一个先进的研究工具，旨在推动可燃冰的探索和利用，通过高效的机器人系统促进科学研究和技术应用。

## 特性
- **高效性**: 机器人系统能够快速高效地进行数据收集。
- **易于使用**: 用户友好的界面，方便科研人员进行操作。
- **扩展性**: 可以根据需要添加新的功能模块。
- **开放数据**: 提供丰富的数据源，支持灵活的数据使用和分析。

## 数据源
数据源位于 `frontend/src/assets/data/content.json`，您可以在此文件中找到平台所使用的所有数据。

## 如何编辑Unity链接
要编辑Unity链接，请直接修改 `content.json` 文件中的相关字段。确保链接格式正确，避免出现错误。

## 运行/构建命令
- **运行平台**: `npm start`
- **构建项目**: `npm run build`

请根据以上信息更新您的操作，此平台将帮助您进行可燃冰研究的各个方面。
